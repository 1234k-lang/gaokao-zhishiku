各省志愿填报特点专辑——全国各省志愿填报政策全解析

写在前面

高考志愿填报政策各省不同，每个省份都有自己的特点和规则。对于2026届考生和家长来说，了解所在省份和目标院校所在省份的具体政策非常重要。我经常遇到这样的情况：考生和家长研究了半天的志愿填报规则，结果发现自己所在省份的政策跟想象的不一样，白白浪费了时间和精力。

这篇文章，我会详细汇总和分析全国各省的志愿填报特点，帮助大家全面了解各省的政策差异，为科学填报志愿做好准备。

第一章：志愿填报模式的省份分布

1.1 采用专业（类）+院校模式的省份

浙江省

浙江省是新高考改革的先行者，采用“专业+院校”模式：

  • 志愿数量：80个
  • 志愿单位：专业（类）+院校
  • 特点：直接填报具体专业，无需考虑校内调剂

山东省

山东省采用与浙江类似的模式：

  • 志愿数量：96个
  • 志愿单位：专业（类）+院校
  • 特点：志愿数量多，选择空间大

河北省

河北省采用“专业+院校”模式：

  • 志愿数量：96个（普通类）
  • 志愿单位：专业（类）+院校
  • 特点：平行志愿，一次性投档

辽宁省

辽宁省也采用该模式：

  • 志愿数量：112个
  • 志愿单位：专业（类）+院校
  • 特点：志愿数量最多，选择最为灵活

1.2 采用院校+专业组模式的省份

湖北省

湖北省采用“院校专业组”模式：

  • 志愿数量：45个
  • 志愿单位：院校专业组
  • 特点：专业组内可调剂

湖南省

湖南省采用类似湖北的模式：

  • 志愿数量：45个
  • 志愿单位：院校专业组
  • 特点：选科要求严格

江苏省

江苏省采用“院校专业组”模式：

  • 志愿数量：40个
  • 志愿单位：院校专业组
  • 特点：历史类、物理类分开

广东省

广东省采用“院校专业组”模式：

  • 志愿数量：45个
  • 志愿单位：院校专业组
  • 特点：本科、专科分别设置

福建省

福建省采用“院校专业组”模式：

  • 志愿数量：40个
  • 志愿单位：院校专业组
  • 特点：提前批设置特殊

1.3 仍采用老高考模式的省份

河南省

河南省采用传统老高考模式：

  • 本科一批：6个院校志愿
  • 本科二批：9个院校志愿
  • 特点：顺序志愿与平行志愿结合

四川省

四川省采用老高考模式：

  • 本科一批：6个院校志愿
  • 本科二批：9个院校志愿
  • 特点：平行志愿在本科二批实施

陕西省

陕西省采用老高考模式：

  • 本科一批：6个院校志愿
  • 本科二批：12个院校志愿
  • 特点：志愿数量较多

安徽省

安徽省采用老高考模式：

  • 本科一批：6个院校志愿
  • 本科二批：10个院校志愿
  • 特点：批次设置较为特殊

第二章：各省份批次设置差异

2.1 批次设置的主要类型

一本二本批次

传统上分为本科一批、本科二批，目前部分省份已经合并。

本科批次合并

越来越多的省份将本科一批和本科二批合并为本科批：

  • 浙江省、山东省、天津市等已合并
  • 河北省、辽宁省等也已合并
  • 这是未来的趋势

提前批设置

各省份提前批设置有所不同：

  • 军事类院校
  • 公安类院校
  • 航海类院校
  • 师范类院校（公费师范生）
  • 医学类院校（免费医学生）
  • 艺术类、体育类

2.2 典型省份批次设置

北京市

  • 本科批：30个院校专业组
  • 专科批：20个院校专业组
  • 提前批：顺序志愿

上海市

  • 本科批：24个院校专业组
  • 专科批：8个院校专业组
  • 提前批：顺序志愿

广东省

  • 本科批：45个院校专业组
  • 专科批：45个院校专业组
  • 提前批：顺序志愿

河南省

  • 本科一批：6个院校志愿
  • 本科二批：9个院校志愿
  • 提前批：顺序志愿

2.3 2026年批次设置变化趋势

本科批次合并加速

预计2026年更多省份将合并本科批次，考生需要关注：

  • 所在省份是否合并
  • 合并后的志愿设置变化
  • 对录取分数的影响

提前批政策调整

部分省份可能调整提前批政策：

  • 提前批招生类型变化
  • 报名条件调整
  • 录取规则优化

第三章：各省份投档规则差异

3.1 平行志愿vs顺序志愿

平行志愿省份

新高考省份和部分老高考省份的本科二批采用平行志愿：

  • 特点：分数优先，遵循志愿，一次性投档
  • 优势：减少高分低就的风险
  • 风险：仍存在滑档和退档风险

顺序志愿省份

部分老高考省份的提前批和本科一批采用顺序志愿：

  • 特点：志愿优先，分数次之
  • 风险：第一志愿非常关键
  • 注意：梯度设置更为重要

3.2 投档比例差异

1:1投档

越来越多的省份实行1:1投档：

  • 特点：招生计划与投档数量相等
  • 优势：减少退档风险
  • 注意：考生需要更精准定位

1:1.05投档

部分省份仍实行1:1.05投档：

  • 特点：有一定数量的机动名额
  • 风险：存在退档可能
  • 注意：建议服从专业调剂

1:1.1投档

部分老高考省份保留1:1.1投档：

  • 特点：投档数量相对较多
  • 风险：退档风险较高
  • 注意：务必勾选服从调剂

3.3 专业录取规则差异

分数优先（分数清）

大多数高校采用：

  • 特点：分数高优先选择专业
  • 优势：对高分考生有利

专业级差

部分高校设置专业级差：

  • 特点：第二专业志愿扣减一定分数
  • 注意：需要考虑级差因素

专业优先（志愿清）

少数高校采用：

  • 特点：第一专业志愿优先
  • 注意：第一专业选择很重要

第四章：各省份加分政策差异

4.1 全国性加分项目

烈士子女

  • 加分幅度：20分
  • 适用省份：全国

归侨、华侨子女、归侨子女

  • 加分幅度：10分
  • 适用省份：全国

台湾省籍考生

  • 加分幅度：10分
  • 适用省份：全国

自主就业的退役士兵

  • 加分幅度：10分
  • 适用省份：全国

4.2 地方性加分项目

少数民族加分

各省份加分幅度不同：

  • 5分、10分、20分不等
  • 需查看具体省份规定

农村独生子女加分

部分省份保留：

  • 5分或10分
  • 需查看具体省份规定

4.3 加分使用规则

加分适用范围

  • 适用于投档
  • 是否适用于专业录取需查看高校招生章程

加分上限

  • 同一考生加分项目不超过20分
  • 累计加分不超过20分

第五章：各省份特殊招生类型

5.1 专项计划

国家专项计划

面向贫困地区招生：

  • 招生院校：中央部门高校和地方重点高校
  • 报名条件：符合当年高考报名条件；本人具有实施区域当地连续3年以上户籍等
  • 录取分数：通常低于普通批分数线

地方专项计划

面向本省农村地区招生：

  • 招生院校：省属重点高校
  • 报名条件：符合当年高考报名条件；本人及父亲或母亲或法定监护人户籍地在实施区域的农村等
  • 录取分数：通常低于普通批分数线

高校专项计划

面向农村学生单独招生：

  • 招生院校：教育部直属高校和其他自主招生试点高校
  • 报名条件：符合当年高考报名条件；本人及父亲或母亲或法定监护人户籍地在实施区域的农村等
  • 录取分数：通常低于普通批分数线

5.2 艺术类招生

统考+校考

考生需参加省级统考和高校校考：

  • 适用于：美术类、音乐类、舞蹈类等
  • 录取规则：文化课成绩+专业课成绩

统考

考生只需参加省级统考：

  • 适用于：部分专业
  • 录取规则：文化课成绩+专业课成绩

5.3 体育类招生

体育单招

运动训练、武术与民族传统体育专业：

  • 报名条件：具有运动训练等项目的二级运动员（含）以上运动技术等级称号
  • 录取规则：文化课成绩+体育专项成绩

体育类专业

体育教育、社会体育指导与管理等：

  • 报名条件：参加省级体育类专业统考
  • 录取规则：文化课成绩+专业课成绩

第六章：各省志愿填报时间差异

6.1 志愿填报时间安排

提前批填报时间

大多数省份在6月下旬：

  • 6月24日-28日
  • 具体时间以各省教育考试院通知为准

本科批填报时间

大多数省份在6月底到7月初：

  • 6月28日-7月5日
  • 具体时间以各省教育考试院通知为准

专科批填报时间

大多数省份在7月下旬：

  • 7月25日-8月初
  • 具体时间以各省教育考试院通知为准

6.2 征集志愿时间

征集志愿时间安排

各省份征集志愿时间不同：

  • 通常在每批次录取结束后2-3天
  • 需要关注省教育考试院通知

征集志愿注意事项

  • 征集志愿是补救机会
  • 关注征集志愿的院校和专业
  • 及时填报

第七章：2026年各省政策变化预测

7.1 可能实施新高考的省份

山西省

山西省可能在2026年实施新高考：

  • 模式可能是“3+1+2”
  • 需要提前了解新高考模式特点

黑龙江省

黑龙江省可能在2026年实施新高考：

  • 模式可能是“3+1+2”
  • 需要提前做好准备

7.2 可能调整志愿设置的省份

增加志愿数量

部分省份可能增加志愿数量：

  • 更好地满足考生需求
  • 提高录取满意度

优化投档规则

部分省份可能优化投档规则：

  • 降低投档比例
  • 减少退档风险

7.3 需要关注的政策要点

加分政策调整

  • 减少加分项目
  • 降低加分幅度

专项计划调整

  • 调整实施区域
  • 优化报名条件

结语

各省高考志愿填报政策存在较大差异，考生和家长需要详细了解所在省份的具体政策。希望这篇文章能够帮助大家全面了解各省的志愿填报特点，为科学填报志愿做好准备。

最后提醒几点：

  1. 以官方信息为准，密切关注省教育考试院发布的最新政策
  2. 了解本省特点，根据本省政策制定志愿填报策略
  3. 提前做好准备，在志愿填报前充分了解目标院校和专业
  4. 注意时间节点，按时完成志愿填报
